Awesome WooCommerce addon, that allows you to customize woocommerce templates/layouts with no coding skills!

= About WooTuner =

WooTuner is a simple to use WooCommerce plugin, that allows to every user, setup WooCommerce layouts:
<ul>
<li>Single Product page</li>
<li>Shop and Product Category pages</li>
<li>WooCommerce breadcrumbs</li>
</ul>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload 'woocommerce-tuner.php' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
